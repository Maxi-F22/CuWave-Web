# CuWave

A just for fun project about a puzzle game where you control a cube. Yup, that’s it. But the cube also has abilities – three, to be precise. Each opposite side has the same ability. Try to reach the goal by using the abilities wisely and positioning the cube correctly.

You can play the game with this link:
